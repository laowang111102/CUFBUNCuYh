# 医疗管理系统 python1299

## 项目概述

医疗管理系统是一款集用户端与管理端为一体的系统。该系统基于Springboot和MySQL技术构建后端服务，同时整合nodejs以提高前端开发的效率与性能。

## 技术栈

### 开发环境

- 操作系统: Windows 10/11
- 数据库: MySQL 8.0

### 后端技术栈

- Web框架: Springboot
- 数据库ORM: MyBatis
- API开发: Springfox Swagger
- 权限管理: Spring Security

### 前端技术栈

- HTML5
- CSS3
- JavaScript
- Bootstrap 4
- jQuery

### 数据库工具

- MySQL Workbench
- Navicat Premium

## 功能模块

系统主要分为以下功能模块：

- 用户端服务
- 管理端操作
  - 病患信息管理
  - 医生与护士管理
  - 药物库存管理
  - 医疗记录管理

## 系统角色

系统主要角色包括：

- 患者
- 医生
- 护士
- 管理员

## 运行环境

- 后端运行环境：Java 11+
- 前端运行环境：现代浏览器

## 部署步骤

1. 配置数据库MySQL 8.0
2. 部署后端服务到Java运行环境
3. 部署前端静态文件到Web服务器

## 核心亮点

- **高效率开发**：基于Springboot和nodejs，提升开发效率。
- **权限管理**：利用Spring Security，实现细致的权限控制。
- **响应式设计**：前端使用Bootstrap，支持多设备访问。
